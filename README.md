# AI Manipulation & Bias Analyzer

This project is an AI-based text analysis system that evaluates digital content to detect emotional influence and potential manipulation patterns.


## Features
- Sentiment Analysis using Transformer-based NLP model
- Emotional influence detection
- Manipulation risk indication
- Interactive web interface using Streamlit

## Technology Stack
Python
Streamlit
Hugging Face Transformers
PyTorch

## How It Works
1. User enters text content.
2. The system processes the text using a pretrained NLP transformer model.
3. Sentiment and emotional intensity are analyzed.
4. The system displays sentiment score and manipulation risk.

## Use Cases
- News analysis
- Social media content evaluation
- Blog and article analysis
- Opinion monitoring

## Deployment
The application is deployed using Streamlit Cloud for easy access and interaction.
